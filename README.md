## About

This repository offers patches for the [Spice2x 'Import from URL'](https://github.com/spice2x/spice2x.github.io/wiki/Patching-DLLs-(hex-edits)#importing-patches-from-a-url) feature.  
For other options, check our [web patcher](https://patcher.moekyun.me/).  

## Usage

For Spice2x patching instructions, see our [patching guide](https://ttwr.moekyun.me/extras/patchsp2x/) and use the URL above.
